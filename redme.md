## Assignment Description

This repository contains C programs related to Stack and Queue data structures. These programs demonstrate the implementation of stack and queue operations using arrays and linked lists. The assignment also includes applications such as infix to postfix conversion, postfix evaluation, and balanced parentheses checking.

All programs are written in C language and compiled using VS Code. Each program follows proper structure, includes comments, and displays output clearly.

---

## Programs Included

### Q1: Stack Operations using Array

This program implements basic stack operations:

* Push
* Pop
* Peek
* Display
  The stack follows LIFO (Last In First Out) principle.

---

### Q2: Queue using Array

This program implements queue operations:

* Enqueue
* Dequeue
* Peek
* Display
  The queue follows FIFO (First In First Out) principle.

---

### Q3: Infix to Postfix Conversion

This program converts an infix expression into postfix expression using stack.
Example:
Input: A+B*C
Output: ABC*+

---

### Q4: Balanced Parentheses

This program checks whether the given expression has balanced brackets using stack.
Example:
Input: {[()]}
Output: Valid Expression

---

### Q5: Postfix Expression Evaluation

This program evaluates a postfix expression using stack.
Example:
Input: 564+*321+*-
Output: Result = 41

---

### Q6: Stack using Linked List

This program implements stack using linked list with operations:

* Push
* Pop
* Peek
* Display

Dynamic memory allocation is used to create stack nodes.
